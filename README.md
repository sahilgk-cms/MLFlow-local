# MLFlow-local



## Trying out MLFlow locally
To run the notebook, please mlflow, scikit-learn first.
Clone the repo and run
```
mlflow ui
```
to view user interface of MLFLow.
MlFLow server will be running on http://127.0.0.1:5000
